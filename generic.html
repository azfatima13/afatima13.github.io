Patient Monitoring System

Collaborated in a team to design and develop a user interface prototype for a patient monitoring device using HTML and JavaScript, focusing on seamless real-time data handling and remote accessibility for hospital personnel to enhance emergency medical response capabilities.

The device we have created is intended to be used in a medical setting. The EKG device will be used so that nurses can store and retrieve patient data, while monitoring their vitals in real time. Our prototype is designed such that the healthcare provider is able to see all relevant information on one screen. The information in the center is specific to the patient, whereas the left and right edges of the device provide useful tools and critical data about the device itself.
